# Portfolio Template

A clean, minimalist personal portfolio template designed to showcase professional skills and projects.

## 💡 Highlights
- **Clean Code**: High readability and organized CSS structure.
- **Static Deployment**: Ready to be hosted on GitHub Pages or Vercel.
- **Customizable**: Easy-to-edit sections for "About Me", "Projects", and "Contact".

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3.
